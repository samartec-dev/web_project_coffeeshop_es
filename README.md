# Triple Espresso

Landing page para una cafetería ficticia (“Triple Espresso”), desarrollada como proyecto de bootcamp en **TripleTen** y pensada para ser el **primer proyecto** de mi portafolio en GitHub.

## Descripción del proyecto

Este sitio presenta una cafetería con enfoque en café de especialidad y una zona de estudio. Incluye una sección principal (hero) con información general (copy de bienvenida, horarios y dirección) y un formulario para **reservar una mesa**.

### Funcionalidad principal

- Página informativa tipo _landing onepage_.
- Navegación simple (por ejemplo, “Reservar una mesa” y “Contactos”).
- Formulario de reserva con campos de:
  - Nombre
  - Número de comensales
  - Fecha y hora
  - Correo electrónico
  - Aceptación de términos de uso

## Tecnologías usadas

- **HTML5** para la estructura del contenido.
- **CSS3** para el diseño y estilos (tipografía, layout, espaciados y componentes visuales).
- Metodología de organización de estilos (por ejemplo, clases reutilizables y estructura ordenada por secciones).

> Nota: Este proyecto está enfocado en maquetación y estilos, priorizando una buena estructura y legibilidad del código.

## Planes de mejora

Estas son ideas para llevar el proyecto a un nivel más cercano a un sitio real (no necesariamente implementadas aún):

1. **Diseño responsive**
   - Adaptar el layout a móviles y tablets (breakpoints, rejillas flexibles, tipografías fluidas).
   - Optimizar el formulario para pantallas pequeñas.

2. **Secciones adicionales de contenido**
   - “Acerca de” (historia de la cafetería, filosofía, origen del café).
   - “Menú” (bebidas, métodos de extracción, alimentos, precios).
   - Información ampliada sobre el café de especialidad y métodos de preparación.

3. **Galería de fotos**
   - Carrusel o grid con fotos del lugar, bebidas, barra y ambiente.
   - Optimización de imágenes (formatos modernos, compresión, carga eficiente).

4. **Newsletter**
   - Suscripción para recibir promociones y novedades (email marketing).
   - Validaciones y mensajes de confirmación.

5. **Blog / Contenido de valor**
   - Artículos sobre café, barismo y métodos de preparación.
   - Contenido informativo: beneficios de socializar, salir de casa y disfrutar un ambiente agradable (con enfoque en información respaldada por estudios cuando aplique).
   - Noticias del negocio (nuevas sucursales, eventos, lanzamientos).

6. **Pedidos a domicilio**
   - Flujo de pedido en línea con selección de productos y checkout.
   - Integración con reparto/zonas de cobertura o plataformas de delivery.
